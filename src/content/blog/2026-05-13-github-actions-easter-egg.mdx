Many years ago I worked on rebuilding the GitHub Actions workflow run overview page. It shows an overview of the different jobs that are part of the workflow run:

![Workflow run page](/assets/posts/2026-05-13-github-actions-easter-egg/easter-egg-highlight.png)

The overview supports zooming and panning, and if you scroll all the way to the left you'll find a small easter egg:

<video controls width="100%" preload="metadata">
  <source src="/assets/posts/2026-05-13-github-actions-easter-egg/easter-egg.mp4" type="video/mp4" />
</video>

I probably hid it too well, and it just says hello and links to [actions/humans.txt](https://github.com/actions/humans.txt).

I'm a biased, I spent a few years working on Actions and you'll find my name in the alumni section, but I still think it's a nice touch and I'd love to see more things like that. Products are built by people, and it's easy to forget that when you're staring at a list of green checkmarks and red X's. Next time you're waiting on a workflow run to finish, scroll left and say hi.
